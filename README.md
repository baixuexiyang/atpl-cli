# atpl-cli
init atpl project   

   
# install  
   
```
npm install -g atpl-cli   
```
    
# start
  
```
mkdir project   
cd project  
atpl init   
```   
   
# options    
  
-h, --help     output usage information    
-V, --version  output the version number   

# commands   
   
init|i   Generate a new project
